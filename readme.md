# mcl for Rust

This is a wrapper library of [mcl](https://github.com/herumi/mcl/),
which is a portable and fast pairing-based cryptography library.

# Test

```
git clone --recursive https://github.com/herumi/mcl-rust
cd mcl-rust
cargo build
cargo test
```
